Site Report

Learning Experience

I learned alot from CST1063 and creating this website. I learnt about html, semantic html and css. There were alot of ups and downs. I especially struggled with understanding animations in css. There were a lot of times where I would have to watch tutorials online to understand some css concepts. There were also plenty of times were my styles just didn't work the way I wanted them to. To solve this I researched online and found that when you right click and press inspect element you can see alot of html & css information about the page. This was a really big help for me. When I finally figured how to create the 'text-highlight' animation I was really happy as I really struggled to do this. Trial and error was typically the way I solved problems as well. Overall the learning experience has been fun, I like seeing my work being displayed visually and so quick in the form of a website!

Design Decisions

Where Design influences came from

I explored colour palettes on coolors.co and found the current colour scheme there. I was going for a dark neon theme, I implemented this by creating the 'text-highlight' class. I found the 'Red Hat Mono' font by looking through google fonts and seeing which one I liked the most.

Creating the Index Name Animation

First, I used a fading transition from a lighter purple, to a darker purple. This is still used in this page, under the class "text-highlight". Then I decided to use a 3D glitch effect, having inspiration from here. To create a similar effect, I decided to use a ::before and ::after pseudoelement. I created animations to move them slightly towards the original word, so it will have a cool seperating effect.

Creating the Navigation Bar

My first step was to create the HTML for the bar. Originally the hamburger menu was a link, however this would not work when i clicked it. I did some research and decided to use a checkbox, as it would work the same. Then I used the :focus selector to check if it was selected, and change the navbar display from none to flex.
